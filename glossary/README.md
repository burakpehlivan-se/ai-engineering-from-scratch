# Sözlük

Gürültüyü kesmek için iki kaynak:

- **[terms.md](terms.md)** - 60+ yapay zeka terimi açıklandı. İnsanların söylediği vs gerçek anlamı.
- **[myths.md](myths.md)** - 20 yaygın yapay zeka yanlış anlaşılmazı, gerçek durumla açıklanıyor.

Her terim girişi şu kalıbı izler:
- **İnsanların söylediği** - Yaygın (genellikle yanlış) anlayış
- **Gerçekte ne anlama geldiği** - Gerçek tanım
- **Neden böyle adlandırıldığı** - Adın kökeni (ilginç olduğunda)
